A propos du site

Bonjour, visiteur ! Je m'appelle Maxime Rossini et te voilà sur mon site personnel.

Ce site est un regroupement de toutes les idées de contenus sympas, de liens utiles et d'outils développés par mes soins. Ce n'est pas un blog, c'est plus une collection de tout ce qui a pu me passer par la tête, que j'ai eu le temps de réaliser et que j'ai trouvé plus pratique à accéder en ligne qu'à conserver sur mon ordinateur.

J'ai eu l'idée de créer le site fin 2013 alors que je quittais mon emploi de développeur et que je souhaitais me former sur une technologie web plus moderne que celles que j'utilisais jusque-là.

Comment naviguer dans le site ?

Les grandes "sections" ou "thèmes" sont accessibles en haut de page dans la bannière du site. L'arborescence de la section actuelle est accessible à gauche dans la zone de navigation.

Infrastructure technique

Voici quelques informations techniques pour les curieux...
Le site est hébergé sur un serveur dédié personnel (chez OVH), et fonctionne grâce aux technologies suivantes :

  • Système d'exploitation : Debian (jessie)
  • Serveur : Apache
  • Contenu dynamique : Django
  • Base de données : PostgreSQL
  • V.C.S. : Mercurial (vous pouvez trouver les sources du site sur BitBucket)
  • Outils divers : South (module Python de migration de base de données pour Django), Psycopg2 (adaptateur PostgreSQL pour Python), Fabric (module Python d'automatisation de déploiement)
